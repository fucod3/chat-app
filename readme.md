chat app

web chat sederhana...

tool yang dipake:
1.termux sebagai server
2.vim sebagai editor teks
3.node js
4.html
5.css
6.js

fitur:
1.bisa chat
2.nggak bisa kirim img/video/audio
3.bisa menambahkan foto profil
4.desain simple


struktur folder:
chat-app/
├── public/
│   ├── index.html
│   ├── app.js
│   └── style.css
├── server.js
├── package.json   
└── .gitignore

cara pake:
1.instal repo ini:
bash
git clone https://github.com/fucod3/chat-app.git
cd chat-app

2.install dependency:
npm install

3. jalanin server:
node server.js

4. buka browser trus masuk ke:
http://localhost:3000
